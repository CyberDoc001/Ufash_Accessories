1. PRODUCT CATALOG FLOW:
   ┌─────────────┐
   │ User clicks │
   │  Catalog    │
   └──────┬──────┘
          │
          ↓
   ┌────────────────────────────────────┐
   │ GET /catalog/ request              │
   └──────┬──────────────────────────────┘
          │
          ↓
   ┌────────────────────────────────────┐
   │ ProductCatalogView processes       │
   │ - Fetch products from DB           │
   │ - Apply filters                    │
   └──────┬──────────────────────────────┘
          │
          ↓
   ┌────────────────────────────────────┐
   │ Render catalog.html template       │
   │ with product data                  │
   └──────┬──────────────────────────────┘
          │
          ↓
   ┌─────────────┐
   │ Display     │
   │ to user     │
   └─────────────┘

2. CHECKOUT FLOW:
   ┌──────────────┐
   │ User clicks  │
   │  Checkout    │
   └──────┬───────┘
          │
          ↓
   ┌──────────────────────────────────┐
   │ CheckoutView processes           │
   │ - Validate stock                 │
   │ - Calculate total                │
   └──────┬───────────────────────────┘
          │
          ↓
   ┌──────────────────────────────────┐
   │ Create Sale record               │
   │ - Store transaction              │
   └──────┬───────────────────────────┘
          │
          ↓
   ┌──────────────────────────────────┐
   │ Create SaleItem records          │
   │ - Store each item sold           │
   └──────┬───────────────────────────┘
          │
          ↓
   ┌──────────────────────────────────┐
   │ Update Product quantities        │
   │ - Reduce stock                   │
   └──────┬───────────────────────────┘
          │
          ↓
   ┌──────────────────────────────────┐
   │ Check for low stock              │
   │ - Create RestockAlerts if needed │
   └──────┬───────────────────────────┘
          │
          ↓
   ┌──────────────────────────────────┐
   │ Generate Receipt View            │
   │ - Display transaction details    │
   └──────┬───────────────────────────┘
          │
          ↓
   ┌────────────┐
   │ Show to    │
   │ user       │
   └────────────┘
